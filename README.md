# Stock Swing Trade Analyzer

This project is a Python stock swing trade analyzer built in Google Colab.

## Features
- Analyzes stock price action using historical daily data
- Calculates 20 EMA, 50 EMA, RSI, volume ratio, support, and breakout levels
- Estimates swing trade setup type
- Generates:
  - Buy Price
  - Stop Loss
  - Sell Target 1 (Short Term)
  - Sell Target 2 (Long Term)
- Uses recent news sentiment in the background
- Shows similar ticker setups for comparison

## Tools Used
- Python
- pandas
- requests
- feedparser
- BeautifulSoup
- trafilatura
- Google Colab
- GitHub

## Notes
This project is designed as a stock screening and swing-trade idea tool. It is not financial advice.
